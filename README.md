# LiDAR Mosquito Defense Project

This project aims to detect mosquitoes and humans using LiDAR technology. It is built with Python and C++ to handle various tasks related to mosquito detection, processing point clouds, and system calibration.

## Features
- LiDAR data acquisition
- Mosquito detection algorithms
- Human avoidance techniques
- Calibration and testing scripts
